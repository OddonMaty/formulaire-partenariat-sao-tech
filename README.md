# Formulaire Partenariat Sao Tech

Ce dépôt contient uniquement le formulaire de candidature partenarial pour Sao Tech Agency.

- Page du formulaire : `formulaire_partenariat_sao_tech.html`
- Page d'accueil GitHub Pages : `index.html` redirige vers le formulaire

## Publication GitHub Pages
1. Créez un dépôt GitHub.
2. Sur votre machine, ouvrez un terminal dans ce dossier.
3. Initialisez Git si nécessaire : `git init`
4. Ajoutez les fichiers : `git add .`
5. Validez : `git commit -m "Ajout formulaire partenariat Sao Tech"`
6. Ajoutez la télécommande GitHub : `git remote add origin https://github.com/<votre-utilisateur>/<votre-repo>.git`
7. Poussez sur `main` : `git branch -M main` puis `git push -u origin main`
8. Activez GitHub Pages sur la branche `main` et le dossier `/`.

L'URL sera : `https://<votre-utilisateur>.github.io/<votre-repo>/formulaire_partenariat_sao_tech.html`
